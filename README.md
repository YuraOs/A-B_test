# A/B Test
## Описание:
Проведен анализ результатов эксперимента. Для эксперимента были задействованы 2 и 1 группы. В группе 2 был использован один из новых алгоритмов рекомендации постов, группа 1 использовалась в качестве контроля. Основная гипотеза заключается в том, что новый алгоритм во 2-й группе приведет к увеличению CTR.
## Стек:
**SQL**, **Pandas** (seaborn, Numpy, scipy, matplotlib).
